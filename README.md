# Kitchen Layout Decision Support MVP

This project is an interactive tool that helps users evaluate and compare kitchen layouts.

## Features
- Adjust position of key elements (fridge, sink, stove, table)
- Evaluate layouts using:
  - Space quality (clearance, walkability, usable space)
  - Workflow efficiency (movement paths)
- Compare baseline vs custom layout

```bash
pip install -r requirements.txt
